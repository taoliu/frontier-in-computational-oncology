# Week 02 image-generation prompts

These prompts are optional visual-draft templates for human lecturers. The main production workflow should focus on text-first slide content and lecturer scripts. Generate images only if needed.

Policy:

- Keep prompts in the public repository.
- Do not commit generated images unless explicitly approved.
- Treat generated images as draft visual placeholders, not final scientific figures.
- Human lecturers may replace generated drafts with their own figures or licensed publication figures.
